# Example Input: PRD Review

Use Role Review Matrix to review this PRD before implementation.

## Product Idea

An AI assistant helps small SaaS founders turn customer feedback into prioritized product decisions.

## Target Users

Solo founders and small product teams.

## Core Workflow

1. Import customer feedback from support tickets and interviews.
2. Cluster feedback into themes.
3. Suggest product opportunities.
4. Draft a short decision memo.

## Decision Needed

Should this move from definition to design?

## Known Constraints

- No customer PII should be stored beyond the imported source.
- The first version should use existing LLM APIs.
- The output must be auditable by the founder.
